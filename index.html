ELO 시스템 설명

ELO(엘로) 시스템은 각 플레이어의 상대 전력을 수치로 표현하는 방식입니다.

  • 초기 ELO: 모든 신규 플레이어는 1000점에서 시작합니다.
  • K 값 (변동 계수):
    • 개인전(1:1): K = 12
    • 팀전(2:2, 3:3, 4:4 등): K = 8
  • 경기 결과에 따라 예상 승률(Expected Score)과 실제 결과(Actual Score)의 차이를 반영해 ELO가 변동됩니다.
    약자가 강자를 이길 경우 상승폭이 크며, 강자가 약자를 이길 경우 상승폭이 작습니다.
  • 팀전의 경우 팀 평균 ELO를 기준으로 상대 팀과 비교하여 승패 결과를 반영합니다.
  • 승/패, 경기 수는 모든 기록실 데이터를 기준으로 자동 집계되며, 이중 계산이나 누락이 발생하지 않습니다.

📈 ELO 계산 예시

아래는 티어 차이에 따른 ELO 변동 예시입니다. (K값은 1:1 = 12, 팀전 = 8 기준)

상황 ELO 차이 예상 승률 승리 시 변화 패배 시 변화
1:1 (동등 실력) 0 50% +6 −6
1:1 (100점 약자 승) −100 36% +8 −4
1:1 (100점 강자 승) +100 64% +4 −8
팀전 (평균 ELO 차 0) 0 50% +4 −4
팀전 (평균 ELO 차 200, 약팀 승) −200 24% +6 −2

※ ELO 변동폭은 상대 실력차가 클수록 비대칭적으로 조정됩니다.
약자가 승리할 때는 상승폭이 커지고, 강자가 패배할 때는 하락폭이 커집니다.
